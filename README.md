# AI Enterprise Knowledge Platform

Skeleton project chạy local bằng Docker Compose.

## Công nghệ

- Frontend: React + TypeScript + Vite
- Backend: FastAPI
- Database: PostgreSQL
- Cache: Redis
- Search storage: Elasticsearch
- Container orchestration: Docker Compose

## Yêu cầu

- Docker Desktop
- Docker Compose v2
- Khuyến nghị Docker có tối thiểu 4 GB RAM
